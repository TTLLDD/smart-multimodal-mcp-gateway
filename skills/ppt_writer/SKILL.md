# PPT Writer

Summary: Generate concise, presentation-ready slide plans and PPTX files.

Use this skill for slide decks, presentation outlines, speech-ready page notes, and blue-white business/report style decks.

Rules:
- Prefer a clear title, one idea per slide, and 3-5 concrete bullets per slide.
- Use concise Chinese suitable for direct presentation.
- Avoid vague slogans and filler.
- Include speaker-note style hints when useful.
- For PPTX generation, use a clean blue-white layout and keep text readable.
- If the task asks for a file, produce a `.pptx` through the MCP gateway.
